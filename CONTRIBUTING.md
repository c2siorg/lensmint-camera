# Contributing to LensMint

Thank you for your interest in contributing to LensMint!

## Getting Started

1. Fork the repository
2. Clone your fork:
```bash
   git clone https://github.com/your-username/lensmint-camera.git
   cd lensmint-camera
```
3. Create a new branch for your changes:
```bash
   git checkout -b feature/your-feature-name
```

## Setting Up Locally

Follow the installation steps in the README to set up the full stack locally.
For contributors without a Raspberry Pi, the hardware-web3-service and
lensmint-public-server can be run independently on any machine.

## Submitting a Pull Request

- Keep PRs focused — one change per PR
- Write a clear PR description explaining what you changed and why
- Reference any related issues using `Fixes #issue-number`
- Make sure your branch is up to date with main before submitting

## Code Style

- JavaScript: follow existing formatting, use `const`/`let` over `var`
- Python: follow PEP 8
- Solidity: follow the existing contract style, use Foundry for tests

## Questions?

Open a Discussion on the repository or reach out via the
[C2SI Slack](https://c2si-org.slack.com).
